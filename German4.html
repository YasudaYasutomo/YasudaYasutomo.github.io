ドイツ滞在の記録(2023年1, 2月)

　現在ドイツの大学院にいます. 内部モデル理論を研究しています.
iphoneで閲覧する場合はリーダーを表示すると読みやすいです.

冬学期も終わり, 休みに入ったのと同時に大学の寮に引っ越しました. 生活が落ち着いたので更新を再開しました. (2023年2月26日).

更新が止まっていた期間は数学とか生活の進捗も停滞していました. ドイツに来てからの半年は身の回りのことで大変と聞いていましたがその通りでした. 大学寮は立地もとてもよく, 過ごしやすいので落ち着いて数学ができています. そろそろドイツに来てから半年が経つのでざっくり振り返ろうと思います. たまに感情的になります.

まずは生活に関してですが住居の問題が常に頭をよぎる半年でした. 大学寮に入れたのは非常に大きなアドバンテージです.

この半年で2回引っ越しをしました. 最初は大学の留学生支援課で紹介してもらったシェアアパートメントに2ヶ月限定で入りました. 先に明らかにしておくと私はシェアパートメントはどちらかといえば避けたい人です. 部屋の清掃に気を遣うのがとてもストレスに感じます. (私のことをよく知っている人は私は掃除をほとんどせず, 部屋が汚いことを知っていると思います. ) あとは単に自分が内向的なだけです. (もちろん外にいるときはパーティに積極的に参加したり, 会話に混じったり, 大学の友人と飲み歩いたりと人と交流することを心がけています. )

結局最初の住居に住んだのは9月の間だけでした. 2ヶ月という短い期間しか滞在場所が確定していないという事実は自分にとって多大なストレスでした. 幸い9月はまだ大学が始まってなかったので家探しにある程度時間が割けました. 結果として少なくとも2023年6月までは住むことのできるシェアアパートメント(2名)を見つけることができそこに滑り込んだのが10月の初めの方でした. 部屋にWi-Fiがないことがとても不満でしたがそうも言ってられないのでテザリングで凌ぐことにしました. 半年以上住むかわからないのに通信環境を整備するのはお金の無駄になりそうだったのでやめました. ルームメイトは学生ではなく働いてるとのことでした. 第一印象は悪くなく, そのときは当面の滞在場所が決まって喜んでいました. 加えてメールやメッセージを大量に送っている最中に訪れた初めての入居チャンスだったのでこれに飛びつくしかなかったと思います. そうして第2の住居での生活が始まりました.

ルームメイトが住居の契約者で彼が私に1室を又貸しするという状況でした. 本来は最初に契約書を残しておくべきでしたが, 彼が仕事のためすぐにドイツを離れるとのことで契約書を書くことはありませんでした. 今思えばここできちんと契約書を作成しておくべきだったと思います. またそのときに新たな住所を登録したので書類の作成をお願いしました. この書類作成に関してはルームメイトが先延ばしをし続けて結局住所登録できたのは11月になってからでした. 滞在許可を取得しないとビザが切れてしまうこともあり非常に焦りました. ルームメイトは明日までに書くと言って結局書かないを何回も繰り返し, そのときは不信感を募らせつつも忙しいのかなと思っていました. 11月末になんとか滞在許可も取れました. ルームメイトはすぐ帰るとは言っていましたが結局12月に2日ほど帰ってきただけでほとんどいませんでした. 正直怪しいなと思って本人に聞いたところ彼の母国で仕事をしており, 今の住居はドイツでの滞在場所を維持するために住み続けているとのことでした. 12月中は特に何もなく年を越しました. 1月頭に市から3ヶ月分の電気代督促が来ました. 光熱費を払っているのはルームメイトだったのですぐに払って欲しいの伝えました. その数日後電気が突然止まりました. ルームメイトに尋ねたところまだ払ってないとのことでした. 驚きました. 結局彼の父親が払いましたが, 5日間電気が使えませんでした. 5日間冷水シャワーを浴びました. ドイツは真冬で気温が0度以下でした. 辛かったです. 体調崩しました. 引っ越すことを決意し動き始めたタイミングで大学寮の入居権利が自分に繰り上がってきて, 即決で入居の手続きをしました. デポジットなどの返金に際しても彼が一度合意した後に言い訳をして支払いを先延ばしにして払わず, その後彼の父親が電話で怒鳴り込んできて合意が破棄されることもありました. (ここカス) 正直精神的に疲弊しておりかつ試験期間中でこれ以上時間を無駄にしたくなかったのでこちらが折れました. 結果としてあちらの支払額が300ユーロほど減りました. ルームメイトがドイツにいないことに加えて, オンラインで送金する手段もないとのことなので彼のガールフレンドが現金を持ってきて鍵を交換になりました. (オンラインで送金する手段あるよね?? その今連絡を取り合っているスマホ使えばできるよね?? 個人的には恋人をそういう風に使うのは理解に苦しみます. その後本人から連絡の一つもない. 謝罪ぐらいあっても良くない?とは思った. ) 第2の住居での生活のあらましはこのような感じでした. 他にも10月中旬に壊れた窓のシャッターの修理が次の年の1月中旬まで先延ばしにされ続けたりしました. (ここもカス) その間真っ暗な部屋で生活していました. 生活リズムこわれる. (あと部屋でタバコを吸うな. )

大学寮に移れたことは非常に幸運だったと思います. このままシェアハウス続行だったら間違いなく体調を崩すか精神的にダメだったと思います. 寮なので部屋そこまで広くありませんが大学院生1人が住むには十分です. また立地がとてもよく, 周りの人にも恵まれています. 今はとても元気です. 数学する余裕も国語を更新する余裕もできました. めでたしめでたし.

26.02.2023

細かく日付を分けて書くと大変なので2月も入れて書きます. ちゃんと更新します. いろいろ機能を実装しようと思います. 静的なページなのでできるかわかりません.

炊飯器をついに買いました. 象印の炊飯器は高いし, 配達に時間がかかるとのことだったので中国の会社が出している炊飯器を買いました. 性能は全く問題ないです. ちゃんと毎日食事を摂ろうと思います. 来週はデュッセルドルフに行くので日本食とキムチを買おうと思います.

3月の中旬にイギリスに行ってきます. 目的はノーサンプトンのファクトリーを巡ることです. 特に今回はエドワードグリーンのファクトリーに行くのが1番の目玉です. あとはジョンロブ, チャーチ, クロケット, トリッカーズ, チーニーのファクトリーに行く予定です. ロンドン市内をがっつり観光するのは今度恋人と行った時にしようと思います. 今回は他の時間はロンドン市内を軽く観光して, ジャーミンストリートに行く予定です. 楽しみすぎる!!!!!

P_maxを勉強した. 記述集合論パートはほとんど全て既知なので他の部分に注力できるおかげでかなりサクサク読み進めることができた.

合うサイズのジャケットの近辺で買うことはほとんど不可能だとわかったので少々お金はかかるがジャケットを仕立てることにした. 紺ブレザーが欲しかったので, 紺ブレザーを仕立てました. 完成するのが楽しみです. ブルックスブラザーズのブラザーと同じくらいの値段で仕立てられるならかなり良いのではないだろうか.

今年の目標ですが, 結果を出すことです. そして論文を書くことです. やるぞやるぞ!!! あとは小さな目標ですが服装の色使いに注意したいと思います. アドバイスあったら欲しいです. 引き続きトラッド継続です.

最近ビーバーがキャベツを運ぶ動画にハマっています. 長浜バイオパークのyoutube動画をみよう.

4ヶ月分の革靴の写真が溜まっていますがアップロードするのが大変なので気になる人(そんな人いるのか??)は私のインスタを見てください. アカントの運用が通常のそれとは大きく外れるので(そのことは私も自覚はしています)フォローは非推奨です, というか革靴写真のアルバムとして使ってるのでフォローしないでください()

妹といとこはそれぞれ無事に大学に合格したそうだ. 安心である. 大学受験が終わってから大学に入学するまでの期間はなんだか不思議な気持ちだった記憶がある. 春の暖かさを感じつつも受験が終わったことでどこか心は涼しくもあった.
